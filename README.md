# しっかリマインダー (MeetingReminder)

## 概要

「しっかリマインダー」は、macOSのメニューバーに常駐するユーティリティアプリです。
カレンダーの予定から次回のWeb会議を自動で検出し、メニューバーから素早く参加できるように支援します。ミーティングの開始が近づくと、全画面の通知でしっかりとお知らせします。

## 主な機能

- カレンダー連携: macOS標準カレンダーと連携し、今日の予定を自動で取得します。
- ミーティングURL検出: 予定の詳細やメモ欄から、ZoomやGoogle MeetなどのURLを自動で探し出します。
- クイックアクセス: メニューバーから今日の予定を素早く確認し、ワンクリックで会議に参加できます。
- 開始前通知: 次のミーティングの開始前に全画面通知を行い、参加を促します。

## 画面構成

本アプリケーションは、主に以下の2つの画面で構成されています。

### 1. クイックメニュー

メニューバーのアイコンをクリックすると表示されるメイン画面です。今日の予定リストの表示、設定、終了などの主要な操作を行います。

### 2. ミーティングアラート

ミーティングの開始直前に表示される全画面の通知画面です。ユーザーに会議が迫っていることを知らせ、即座に参加できるようになっています。

## プライバシーについて

このアプリケーションは、ユーザーのプライバシーを尊重します。

- アクセス範囲: macOSに標準で搭載されているカレンダーアプリのデータにのみアクセスします。
- 利用目的: カレンダーから取得する情報は、次のミーティング時刻を計算し、通知をスケジュールする目的のみに利用されます。
- 情報の非保持・非送信: 予定のタイトル、参加者、メモといった、通知の目的以外のユーザーの情報を、アプリケーションの外部に送信したり、保存したりすることは一切ありません。すべての処理はコンピュータ内で完結します。
